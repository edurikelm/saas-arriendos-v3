# Prototypes

Carpeta para almacenar prototipos experimentales de componentes y funcionalidades.

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `property-card-prototypes.tsx` | Prototipos de tarjetas de propiedades |
| `reservation-card-prototypes.tsx` | Prototipos de tarjetas de reservas |
| `reservation-form-prototypes.tsx` | Prototipos de formularios de reservas |

## Notas

- Estos archivos son experimentales y no deben usarse en producción
- Cada archivo contiene variantes/prototipos de su componente final
- Una vez validado un diseño, mover a su ubicación definitiva en `src/components`
