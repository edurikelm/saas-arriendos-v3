# Triage labels

This project uses the canonical five-label triage vocabulary.

| Label | Meaning |
|-------|---------|
| `needs-triage` | New issue, awaiting evaluation |
| `needs-info` | Waiting on reporter for more info |
| `ready-for-agent` | Fully specified, AFK agent can pick it up |
| `ready-for-human` | Needs human implementation |
| `wontfix` | Will not be actioned |
